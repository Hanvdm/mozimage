mozImage 1.3 beta

The main effort of this release will be integrate a simple module called mozimage_edit and to improve the user interface.

MozImage edit

This module was developed by My Hoang and give you the ability to modify images throught ImageMagick. So you must first install ImageMagick then open the options panel of MozImage and fill the voice named "Path to conver" with the main executable of ImageMagick, for examples: 

C:\Program files\ImageMagick6\convert.exe

User interface

In this release we try to give to MozImage an interface more consistent with the rest of the browser, so now MozImage is shown as a side bar like the bookmark bar or the history bar.
