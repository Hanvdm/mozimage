mozImage 1.1 beta

This beta release added the possibility to create simple albums and to scan web pages looking for images.

Bookmarks

The bookmarks concept is much like that of the browser: when you are seeing the images of a folder you can click on "add bookmark" button and that folder goes to your bookmarks.

The bookmarks are stored on a file named mozimage-bookmarks.txt on your profile directory with name and directory separated with a new line.

Web page analysis

In this release there are two ways to open mozImage: through the tools menu and with the context menu of the current web page. With this last method mozImage scan the current page in search of links that point to an image. If the link itself is an image mozImage display the image on the left of the link name (you must enable the thumbnails on the options panel).
